# User Registration Form Project

## Description
This project demonstrates a simple user registration form using Node.js, Express, and MySQL. Users can register by providing a username, email, and password, which is stored in a MySQL database.

## Features
- HTML form for user registration.
- Node.js server with Express.js to handle form submission.
- Data inserted into MySQL using prepared statements for security.
- Feedback to the user upon successful registration.

## Technologies Used
- Node.js
- Express.js
- MySQL
- HTML
